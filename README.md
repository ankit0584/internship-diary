# 📓 Internship Diary — Frontend

React + Vite frontend for the Internship Diary web app.

## Tech Stack
- React 18
- Vite
- Axios (API calls)
- React Router DOM (navigation)

## Pages
- `/login` — Login page
- `/register` — Register page
- `/student/dashboard` — Student diary dashboard
- `/staff/dashboard` — Staff review dashboard

## Setup
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5173`

> Make sure the backend API is running on `http://localhost:5283` before starting the frontend.
